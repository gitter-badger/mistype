Mistype

Mistype is here to help you.

How many times did you notice error on site and couldn't inform author, because you didn't know how?

But if author had integrated Mistype, you could just select wrong text and hit Ctrl+Enter. It's that simple.

How to enable Mistype on my site?

After one-step registration you'll receive a little <script> tag. Just drop it in your site's code and you're covered. Be sure to insert some text like “Report mistakes via Ctrl+Enter” somewhere as well to notify users about their's new superpower.

How would I receive notifications?

We'll send you an email for each reported error.

URL of page, class list and tag name of element with mistyped text are remembered, so you can track it back.

How do I customize text of notification?

Insert data-text attribute at <script> tag, like < script="//www.mistype.co/js" data-typo data-owner="your_api_key" data-text="That's cool, thanks">. Don't forget to escape any " or ' inside of the string.

Something else?

You can add any additional info to request: just place function name in data-additional. Our JS is checking whether is window[data-additional-value] a function, and if so, executes and attaches return value. Otherwise it's value is simply copypasted to request (it can be helpful if you generate markup on server, you could place username in this field, for example).

Appearance of notifications can be styled with .mistype-notification and .mistype-notification.in-action classes, first is generic, second means transition (in & out).
